# Health Records Dashboard

## Project Overview
This project analyzes a Health Records dataset using charts, graphs, and KPI cards to generate meaningful healthcare insights.

## Tools Used
- Excel / Power BI
- GitHub
- PowerPoint

## Dataset Columns
- Name
- Age
- Gender
- Blood_Group
- Medical_Condition
- Test_Result
- Billing_Amount

## Dashboard Features
- Total Patients KPI
- Average Age KPI
- Total Billing KPI
- Age Group Analysis
- Gender Distribution
- Blood Group Count
- Medical Condition Analysis
- Billing Analysis
- Test Result Distribution

## Key Insights
- Most patients belong to adult and senior age groups.
- Common medical conditions identified.
- Billing trends visualized.
- Test results summarized clearly.
